El código adjunto es el encargado de dibujar los tres tipos de fractales implementados: Koch, sierpinsky y sierpinsky carpet.
Dentro del código existen unas variables globales (hacerKoch, hacerSierpinsky, hacerSierpinskyCarpet) que permiten alternar cual sera el fractal a dibujar.

Para el desarrollo de esta práctica se ha vuelto a usar la librería Turtle, que permite un dibujo animado, dando un resultado interesante a la práctica.
